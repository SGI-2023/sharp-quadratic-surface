Get the one ring of a vertex.

The one ring of both faces and vertices counter clockwise around the vertex are returned. For boundary vertices, the faces and vertices start at the right boundary and traverse the faces in order to the left boundary. For interior vertices, an arbitrary start face is chosen, and the vertex one ring is closed so that v_0 = v_n.
