Detailed Description

Methods to construct and evaluate polynomials in one variable. We represent polynomials as column vectors of coefficients with basis order 1, t, t^2,.... For vector valued polynomial functions (i.e. functions f: R -> R^n where each coordinate projection is a polynomial), we use matrices with the column vectors as columns. We refer to these vector valued polynomial functions as polynomial mappings to distinguish them from scalar valued polynomials. Evaluation of the polynomials and polynomial mappings is then simply the left product of the row vector [1 t t^2 ... t^n] with the column vector or matrix.
