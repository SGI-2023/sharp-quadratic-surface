Detailed Description

Local layout manifold chart in R2 of the one ring around a central vertex.

The one ring is represented as a sequential list of the n faces and n + 1 vertices around the central vertex along with the corresponding vertex uv positions. For i = 0,...,n-1, the ith vertex corresponds to to the first vertex in face i ccw from the central vertex.

For an interior vertex, the one ring begins at an arbitrary face, and the nth vertex is the same as the first. For a boundary vertex, the one ring begins as the boundary face to the right of the vertex and ends at the left boundary face, and the nth vertex is generally different from the 0th.
