\doxysubsection{Detailed Description}
A quadratic surface patch with convex polygonal domain.

Supports\+:
\begin{DoxyItemize}
\item evaluation and sampling of points and normals on the surface
\item triangulation
\item conversion to Bezier form
\item bounding box computation
\item boundary curve parameterization
\item cone point annotation
\item (de)serialization 
\end{DoxyItemize}
